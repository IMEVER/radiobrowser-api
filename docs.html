List of countries
Syntax:
http://www.radio-browser.info/webservice/json/countries
http://www.radio-browser.info/webservice/json/countries/<filter>
http://www.radio-browser.info/webservice/xml/countries
http://www.radio-browser.info/webservice/xml/countries/<filter>
Description:
Will get a JSON encoded list of all countries in the database. If a filter is given, it will only only return the ones that contain the filter as substring.
Supported output formats: json,xml
Example:
http://www.radio-browser.info/webservice/json/countries
http://www.radio-browser.info/webservice/xml/countries/aus

List of radio stations
Syntax:
http://www.radio-browser.info/webservice/xml/stations
http://www.radio-browser.info/webservice/xml/stations/byid/searchterm
http://www.radio-browser.info/webservice/xml/stations/byname/searchterm
http://www.radio-browser.info/webservice/xml/stations/bynameexact/searchterm
http://www.radio-browser.info/webservice/xml/stations/bycodec/searchterm
http://www.radio-browser.info/webservice/xml/stations/bycodecexact/searchterm
http://www.radio-browser.info/webservice/xml/stations/bycountry/searchterm
http://www.radio-browser.info/webservice/xml/stations/bycountryexact/searchterm
http://www.radio-browser.info/webservice/xml/stations/bystate/searchterm
http://www.radio-browser.info/webservice/xml/stations/bystateexact/searchterm
http://www.radio-browser.info/webservice/xml/stations/bylanguage/searchterm
http://www.radio-browser.info/webservice/xml/stations/bylanguageexact/searchterm
http://www.radio-browser.info/webservice/xml/stations/bytag/searchterm
http://www.radio-browser.info/webservice/xml/stations/bytagexact/searchterm

http://www.radio-browser.info/webservice/json/stations
http://www.radio-browser.info/webservice/json/stations/byid/searchterm
http://www.radio-browser.info/webservice/json/stations/byname/searchterm
http://www.radio-browser.info/webservice/json/stations/bynameexact/searchterm
http://www.radio-browser.info/webservice/json/stations/bycodec/searchterm
http://www.radio-browser.info/webservice/json/stations/bycodecexact/searchterm
http://www.radio-browser.info/webservice/json/stations/bycountry/searchterm
http://www.radio-browser.info/webservice/json/stations/bycountryexact/searchterm
http://www.radio-browser.info/webservice/json/stations/bystate/searchterm
http://www.radio-browser.info/webservice/json/stations/bystateexact/searchterm
http://www.radio-browser.info/webservice/json/stations/bylanguage/searchterm
http://www.radio-browser.info/webservice/json/stations/bylanguageexact/searchterm
http://www.radio-browser.info/webservice/json/stations/bytag/searchterm
http://www.radio-browser.info/webservice/json/stations/bytagexact/searchterm
Description:
Will get a list of all radio stations. It is possible to add a search term to only get stations that have the term in their names. The variants with "exact" will only search for perfect matches.
It is also possible to select stations based on their country, state, language, tags or id.
Supported output formats: json,xml
Parameter:
This parameters can be added by using a HTTP POST request with the encoding "x-www-form-urlencoded" instead of GET request.
name default value possible values description
order name name, url, homepage, favicon, tags, country, state, language, votes, negativevotes, codec, bitrate, lastcheckok, lastchecktime, clicktimestamp, clickcount, clicktrend name of the attribute the result list will be sorted by
reverse false true, false rerverses the result list if set to true
offset 0 0,1,2,3,4,.. starting value of the result list from the database, for example if you want to do paging on the serverside
limit 25 0,1,2,.... number of returned datarows (stations) starting with offset
Example:
http://www.radio-browser.info/webservice/xml/stations
http://www.radio-browser.info/webservice/json/stations/byname/jazz
http://www.radio-browser.info/webservice/xml/stations/bycountry/austria
http://www.radio-browser.info/webservice/xml/stations/bycountryexact/austria

Playable station url
Syntax:
http://www.radio-browser.info/webservice/xml/url/stationid
http://www.radio-browser.info/webservice/json/url/stationid
http://www.radio-browser.info/webservice/pls/url/stationid
Description:
Returns the playable url for the station. Does return 404 code if the playlist on the remote server could not be retrieved. In this case the station is not working any more and needs to get updated. Supported output formats: json,xml,pls (playlist)
Example:
http://www.radio-browser.info/webservice/xml/url/123
http://www.radio-browser.info/webservice/json/url/123
http://www.radio-browser.info/webservice/pls/url/123

Vote for station
Syntax:
http://www.radio-browser.info/webservice/xml/vote/stationid
http://www.radio-browser.info/webservice/json/vote/stationid
Description:
Increase the vote count for the station by one. Can only be done by the same ip adress for one station every 10 minutes. Returns the changed station as result if it worked.
Supported output formats: json,xml
Example:
http://www.radio-browser.info/webservice/json/vote/123
